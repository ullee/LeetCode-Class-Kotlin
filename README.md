# LeetCode-Class-Kotlin
LeetCode Class in kotlin

## ListNode
### Initialize from Collection's List
```kotlin
val listNode = ListNode.from(listOf(1, 2, 3))
```

### Display
```kotlin
println(listNode)
// 1 -> 2 -> 3
```

### ListNode conversion to Collection's List
```kotlin
val list = listNode.toList()
// [1, 2, 3]
```
